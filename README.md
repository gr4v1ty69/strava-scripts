# Strava Custom Scripts

Two userscripts that enhance your Strava experience: one unlocks premium route planning features for free, and the other adds beautiful custom map layers to your activity pages.

### Strava Route Planner for All
Unlock Strava's premium route planning features without a subscription. Create and edit routes with all the tools normally reserved for paid members.

### Strava Custom Maps
Transform your activity maps with 9 different map styles including topographic, cycling-focused, winter, satellite, and hiking trail overlays. Perfect for analyzing your routes with specialized maps.

1. **Install a userscript manager** (one-time setup):
   - [Tampermonkey](https://www.tampermonkey.net/) - Works on Chrome, Firefox, Safari, Edge

   - Click here for : [strava-custom-maps](https://cdn.jsdelivr.net/gh/gr4v1ty69/strava-scripts@main/strava-custom-maps.user.js)
   - Click here for : [strava-route-planner](https://cdn.jsdelivr.net/gh/gr4v1ty69/strava-scripts@main/strava-route-planner-for-all.user.js)

That's it! The scripts will automatically activate when you visit Strava.

## ğŸ¯ Usage

### Route Planner
Simply visit any Strava route page and start planning - premium features are now unlocked:
- https://www.strava.com/routes/new
- https://www.strava.com/maps/create

### Custom Maps
Open any activity page and look for the map layer switcher in the top-right corner:

**Available Maps:**
- OpenStreetMap - Standard street map
- OpenTopoMap - Detailed topographic view
- CyclOSM - Cycling infrastructure highlighted
- mapy.cz Outdoor - Excellent European outdoor map
- mapy.cz Winter - Winter sports trails
- mapy.cz Aerial - Satellite imagery
- mtbmap.cz - Mountain biking trails (Europe)
- freemap.sk - Outdoor trails
- Esri Satellite - High-quality satellite view
- Waymarked Trails - Hiking trail overlays

The blue dot automatically syncs with your activity playback!

## ğŸ”§ Troubleshooting

**Route planner not working?**
- Make sure you're on a Strava route planning page
- Check that the script is enabled in Tampermonkey

**Maps not showing?**
- Verify you're on an activity page (not route planning)
- Try refreshing the page
- Some map providers load slower than others
